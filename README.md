# 🚀 Git Commit Helper

`chel` (Commit Helper) は、対話式で Git のコミットメッセージを作成する CLI ツールです。

## 📦 インストール手順

1. **リポジトリをクローン**

   ```sh
   git clone https://github.com/your-username/git-commit-helper.git
   cd git-commit-helper
   ```

2. **依存パッケージをインストール**

   ```sh
   npm install
   ```

3. **セットアップ（シンボリックリンクを作成）**

   ```sh
   bash setup.sh
   ```

4. **コミットヘルパーの使用**
   ```sh
   chel
   ```
   または
   ```sh
   commit-helper
   ```
